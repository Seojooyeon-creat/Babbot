https://github.com/Seojooyeon-creat/Babbot

## 📝 소개

기분·날씨·상황을 **자유 텍스트로 입력**하면 Claude AI가 그에 맞는 메뉴를
추천 이유, 먹는 팁과 함께 알려주는 채팅형 웹 앱입니다.

## 🛠 사용 방법

1. 위 배포 주소 접속
2. 하단 입력창에 상황을 자유롭게 입력
   - 예: `비 와서 우울한데 혼자 따뜻한 거 먹고 싶어`
3. AI가 메뉴를 추천해줌 (대화를 이어가며 더 구체적으로 추천받을 수 있음)

## 🧩 구성

| 파일 | 역할 |
|------|------|
| `app.py` | Streamlit 웹 프론트엔드 (채팅 UI) |
| `recommender.py` | 공용 핵심 로직 (시스템 프롬프트 + Claude API 호출) |
| `main.ipynb` | CLI 버전 (노트북, 같은 로직 사용) |
| `requirements.txt` | 의존성 목록 |

## ⚙️ 기술 스택

- Python / Streamlit (웹 UI)
- Anthropic Claude API (`claude-haiku-4-5`)
- Streamlit Community Cloud 배포

> API 키는 코드에 포함되지 않으며, 배포 서버의 Secrets에만 안전하게 저장되어 있습니다.
